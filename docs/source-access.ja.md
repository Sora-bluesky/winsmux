# 公開配布の境界

このページは、`v1.0.0` 系から適用する公開配布の境界を説明します。

## 公開配布

`v1.0.0` 以降の winsmux は、完全な実装ソース一式ではなく、リリース済みの製品として公開配布します。公開リリースに含めるものは次の通りです。

- Windows デスクトップインストーラー
- MSI 配布物
- npm パッケージと CLI インストール経路
- チェックサムファイル
- リリースノート
- 利用者向けとコントリビューター向けの公開ドキュメント
- バグ、サポート、機能要望のための公開 issue

公開リポジトリには、ドキュメント、リリースメタデータ、例、セキュリティ方針、互換性メモを残せます。公開ドキュメントには、非公開のソース共有、レビュー、事業上の取り決めを書きません。

## 境界

公開利用者は、完全な実装ソースへのアクセスなしでインストール、更新、検証、issue 報告ができます。公開リリースには、winsmux を再配布したり、競合する公開ソースミラーを作ったりする権利は含みません。

セキュリティ報告には公開のセキュリティ方針を使ってください。秘密情報、資格情報、顧客の非公開データを公開 issue に送らないでください。
